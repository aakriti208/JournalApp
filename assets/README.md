# Assets Directory

Place your app assets here:

## Required Assets

1. **icon.png** (1024x1024)
   - App icon for iOS and Android

2. **splash.png** (1284x2778)
   - Splash screen image

3. **adaptive-icon.png** (1024x1024)
   - Android adaptive icon foreground

4. **favicon.png** (48x48)
   - Web favicon

## Generating Assets

You can use Expo's asset generation:

```bash
npx expo prebuild
```

Or create them manually and place them in this directory.

## Temporary Placeholder

Until you create custom assets, you can use the default Expo assets or create simple colored squares:
- icon.png: Blue square with white "J" initial
- splash.png: White background with app name
- adaptive-icon.png: Same as icon.png
- favicon.png: Smaller version of icon.png
